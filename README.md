# Learnlytics

Aplicación móvil para la gestión y análisis de hábitos de estudio mediante inteligencia artificial.

## Tecnologías

- Frontend: Ionic + Angular
- Backend: Node.js + Express
- Base de datos: MongoDB
- IA: API externa (LLM)

## Objetivo

Permitir a estudiantes registrar sus hábitos de estudio y recibir recomendaciones personalizadas basadas en análisis de datos.
